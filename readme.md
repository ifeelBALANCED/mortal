# Getting started
1. **Copy .env and adjust env vars if necessary**

```bash
$ cp packages/frontend/.env.example packages/frontend/.env.local

```

2. Install dependencies.

```bash
$  yarn
```

3. Start dev server with hot reload at http://localhost:3000.

```bash
$  yarn run dev
```
